# Pandauth

Pandauth is a personal proyect thath consist a basic login without db typically known like **mysql**, **Mariadb**, etc...
in this case I am using a spreadsheet.

## Requeriments

```bash
$ pip3 install pandas
$ pip3 install getpass
```
